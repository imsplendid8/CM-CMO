# 도구별 상세

공통: 좌측 사이드바에서 상품 선택, 상단 테마 토글. 상품 집합은 5종 모두 동일(홈·주택화재·골프·암·치아·운전자·여성건강·태아·해외여행·해외장기체류).

## 1. 테크니컬 SEO 콘솔 — `seo-audit.html`
- **목적**: 담당 상품 현행 화면(제목·메타·OG·H·robots·분기)을 진단하고, 화면에 바로 붙일 HTML 태깅 가이드·robots.txt 제공.
- **핵심**: 상품별 우선순위 To-Do + before/after 코드, 네이버 웹마스터도구 가이드 반영, Excel/PPT 전달용 산출.

## 2. 검색광고 키워드 추출 — `keyword-tool.html`
- **목적**: 상품 시드 × 수식어로 검색광고 키워드 대량 생성·분류·CSV.
- **데이터**: `PRODUCTS`(상품)·`MODS`(수식어)·`AGES`. `genKeywords()`가 거래/정보/브랜드/담보/AI 인텐트로 분류하고 플랫폼(네이버/구글)·매치타입 추천.
- **네이버 실시간 검색량**: `livePanel`(🔌 네이버 실시간 검색량)에서 상품 선택 후 버튼 하나로 검색광고 API 월검색량·경쟁도를 프록시(`/searchad/keywordstool`) 경유로 병합(팀 공유 프록시라 키 입력 불필요, 개인 키 override 가능). 로컬 서버 폴백: `scripts/naver_local_server.py`. `docs`의 KEYWORD-API 참조.

## 3. 카테고리 뉴스 모니터링 — `news-tool.html`
- **목적**: 상품과 같은 카테고리 기준 경쟁·시장 뉴스 요약·시사점.
- **데이터**: `NEWSMON[상품키] = { summary, insight, items:[{t,src,date,url,gist}], live? }`. 헤드라인·링크는 공개 뉴스, 요약·시사점은 자체 분석.
- **실시간 갱신(선택)**: 로컬서버 `POST /news`(네이버 개발자센터 검색 API 프록시) → 🔄 실시간 갱신.

## 4. 검색결과 주간 아카이브 — `serp-tool.html`
- **목적**: 상품 대표키워드(PC) 네이버 검색결과를 매주 스크린샷으로 아카이빙 → 전/후 비교로 변화 추적.
- **아카이브**: 파일 선택/드래그 → 이미지 다운스케일 후 `localStorage(mf_serp)`에 날짜별 저장(전송 0). ⬇백업/⬆복원 JSON.
- **분석**: `SERPINTEL[상품키] = { comps:[{who,area,hook}], ours, change }` — 경쟁사 소구포인트(네이버 `search_webkr` 근거)·우리 대응·변화. 한화 상위노출 갭 표시(주택화재·골프·해외여행).
- **전/후 시각 diff**: 두 주차를 고르고 **🔍 변화 감지** → canvas 픽셀 비교(pixelmatch 방식)로 바뀐 영역을 붉게 하이라이트 + 변화율%·민감도 슬라이더·길이변화 경고(외부 라이브러리 0).
- **참고**: 자동 캡처는 네이버 봇차단으로 불가 → 수동 아카이브가 정식 경로.

## 5. 연간 시즈널 이슈 캘린더 — `seasonal-tool.html`
- **목적**: 상품별 계절 이슈·키워드·**선제 대응 시점**(예: 주택화재 6~8월 장마 → 누수·침수·풍수재).
- **데이터**: `SEASONAL[상품키] = [{ m:[월], tag, why, kws:[], act }]`.
- **뷰**: 이번 달 주목 + 다음 달 선제대응 예고, 상품×월 히트맵, 시즌별 카드, 키워드 칩(네이버 검색·데이터랩 검증).

## 분석 스킬 — `.claude/skills/cm-news-analysis/`
뉴스·SERP·시즌 분석 산출물의 톤·포맷 표준. 핵심요약+상세 2단계, 이슈–영향–참고 3요소, 메인 3종 강조. `NEWSMON`·`SERPINTEL`·`SEASONAL` 갱신에 사용.
