# 검색광고(SA)·브랜드검색(BSA) 솔루션 로드맵

_각 기능을 "목표·입력·출력규격·연동·제약·검증" 6요소로 정의해 하나씩 구현. 광고 문구·소재는 모두 **광고심의 검수 전제(초안)**, 데이터는 **공개·비식별**, 키는 **워커 시크릿**._

## A. 검색광고(SA) 솔루션

| # | 기능 | 상태 | 필요 입력 |
|---|---|---|---|
| A1 | 키워드 추출 → **네이버 대량등록 파일**(광고그룹핑) | ✅ 완료 | — |
| A2 | 검색광고 **문구 소재** 제작 (제목·설명 N종) | ✅ 완료 | — |
| A3 | 검색광고 **파워컨텐츠 소재** 제작 | ⬜ | 파워컨텐츠 규격(제목/본문/이미지 슬롯) |
| A4 | **소재 규격 → 딸깍 생성** (시즌캘린더·뉴스·SERP 아카이브 연동) | ⬜ | 규격 스키마 |
| A5 | **심의 규격 → 텍스트+썸네일 시안**(썸네일은 SVG 목업) | ⬜ | 심의 체크리스트 |
| A6 | 네이버 API로 **그룹별 예산 on/off·소진 현황** | ⬜ | 검색광고 **관리 API** 권한(검색 API와 별개) |

### A1 상세 (완료)
- `keyword-tool.html` → **네이버 대량등록** 버튼: 캠페인=상품 / 광고그룹=상품×인텐트(거래·브랜드·담보·정보), 네이버 대상만(구글·AI 제외).
- 컬럼: 캠페인유형·캠페인·광고그룹·키워드·입찰가(빈칸)·연결URL(PC/모바일)·인텐트·월검색량·경쟁도.
- 실제 네이버 대량관리 템플릿 컬럼을 주시면 그 규격으로 정밀 매핑 가능.

### A2 상세 (완료)
- `adcopy-tool.html` (허브 6번째 뒤, ✍️ 검색광고 소재): 상품별 **제목(≤15자)·설명(20~45자) 후보 자동 생성** + **직접 작성 실시간 글자수·심의 표현 체크**.
- 규격: 네이버 파워링크(제목 15자 / 설명 20~45자, 특수문자·최상급 자제). 소구 토큰(USP·담보·시즌)으로 후보 조합, 규격 밖 후보는 자동 제외.
- 금지·주의 표현 린트(최고·최저가·1위·무조건·100%·확정·무심사 등) → 보험 광고심의 전제(초안).
- 심의 세부 규칙(금지/필수 문구)을 주시면 BAN/필수 체크를 그 기준으로 정밀화 가능.

## B. 브랜드검색(BSA) 솔루션

| # | 기능 | 상태 | 필요 입력 |
|---|---|---|---|
| B1 | 네이버 **브랜드검색 소재 모니터링** | 🔶 부분(경쟁사 브랜드검색 캡쳐 있음) | 자사 BSA 소재 기준 |
| B2 | 네이버검색 API로 **BSA on/off·계약현황 점검** | ⬜ | 노출 판정 기준(캡쳐/검색 기반), (가능 시) 계약 데이터 |
| B3 | **BSA 사용 키워드 제안** + **3개월 재계약 판단**(단가 구간) | ⬜ | **BSA 단가 구간표**, 상품별 검색량 |

## 제약·현실 체크
- **문구·소재**: AI는 **초안**만 — 최종은 광고심의·법무 검수.
- **썸네일**: SVG/canvas **목업** 수준(실제 광고 이미지·디자인 아님).
- **예산 on/off·계약현황**: 네이버 **검색광고 관리 API**(및 BSA 관련 권한)가 있어야 실데이터. 없으면 검색결과 노출 기반 간접 판정 + 샘플 로직.
- 단가표·심의 기준은 **사내 자료** → 프롬프트에 붙여주면 그 기준으로 구현.
