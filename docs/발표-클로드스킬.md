# 클로드 스킬로 마케팅 업무 자동화 — 적용 전후

_한화손보 장기CM · Modooflow · 발표 요약_

---

## 1. 클로드 스킬이란?

반복 업무의 **① 방법(절차) · ② 판단 기준 · ③ 어투·포맷**을 하나로 패키징해 두면,
관련 요청이 들어올 때 AI가 **매번 같은 품질·같은 형식**으로 수행합니다.

> 쉽게: "우리 회사 담당자가 일하는 방식"을 AI에게 **한 번 가르쳐 자산화** → 누가 시켜도 그 기준대로.

- 특정 키워드/맥락이 나오면 **자동 적용**(트리거).
- 사람이 매번 프롬프트를 길게 쓸 필요 없음 → **일관성 + 속도 + 온보딩 절감**.

---

## 2. 우리가 만든 스킬 3종 (Modooflow)

| 스킬 | 하는 일 | 트리거 예시 |
|---|---|---|
| **cm-news-analysis** | 뉴스·경쟁사 동향을 **담당자 관점**으로 분석(핵심요약→상세, 이슈·영향·참고 3요소, 메인3종 강조) | "뉴스 분석", "타사 동향", "경쟁사" |
| **card-news-summary** | 헤드라인 묶음 → **카드뉴스**(제목·요지·담당자 액션 3줄) | "카드뉴스", "헤드라인 정리" |
| **insurance-terms** | 표준약관 어려운 용어 → **소비자 쉬운 표현·가입 카피**(난이도·심의 전제) | "약관 용어", "쉬운 표현", "가입 카피" |

→ 실제로 뉴스 툴·약관 툴·SERP 분석에 연결되어 **산출물의 톤·포맷을 표준화**.

---

## 3. 적용 전 → 후 (기대효과)

| 업무 | 적용 전 | 적용 후 |
|---|---|---|
| **뉴스 브리핑** | 수동 검색·선별·정리 **회당 30~60분**, 담당자마다 품질 편차 | **네이버검색 MCP + 스킬**로 수 분 내, 이슈·영향·참고 표준 포맷, 메인3종 자동 비중 |
| **경쟁사 동향** | 산발적·주관적 | 손보 **빅4/5 표준 요약·시사점** 일괄 |
| **약관 카피** | 감으로 순화, 용어 오남용·심의 리스크 | **표준약관 근거 + 난이도 등급 + 심의 전제**, 표현을 **재사용 자산화** |
| **카드뉴스** | 기획·디자인 수작업 | 헤드라인만 넣으면 **카드 초안 즉시** |
| **신규 담당자** | 기준·톤 익히는 데 시간 | 스킬이 **기준을 내재화** → 바로 표준 품질 |

### 정량 기대효과 (추정)
- 뉴스·동향 리서치 시간 **약 70~80% 단축**
- 산출물 **품질 편차 최소화**(표준 포맷·기준 고정)
- 신규 담당자 **온보딩 기간 단축**(스킬 = 살아있는 업무 매뉴얼)
- 약관 카피 **심의 리스크·재작업 감소**

---

## 4. 확장성 — 스킬은 계속 쌓인다

- 다음 후보: **브랜드검색 계약 판단**(단가표 기반), **SEO 카피 생성**, **타사 브랜드검색 분석**
- **MCP 결합**: 네이버 검색 MCP로 수집 → 스킬로 분석 → 대시보드 반영 = **수집·분석 반자동화**
- 스킬은 **버전 관리·공유**되어 팀 전체가 같은 자산을 사용

---

## 5. 리스크·거버넌스 (반드시 병기)

- **공개 데이터만** 사용 — 고객·임직원 개인정보·영업비밀은 입력 금지(가상·비식별).
- 광고성 문구는 **광고심의 검수 전제** — AI 산출물은 **초안**, 최종 판단·검수는 **사람**.
- 약관 순화 표현은 **면책·감액 등 불리 조건을 정확히**(오인 소지 배제).

## 6. 한 줄 결론

> **"담당자의 일하는 방식을 AI 자산으로 만들면, 반복 리서치·정리는 자동화하고 사람은 판단·전략에 집중한다."**
> 이미 3개 스킬이 뉴스·약관·카드뉴스에 적용 중이며, 붙일수록 효과가 누적됩니다.
